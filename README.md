# Cat and Dog Image Classification Project

## Description
This project classifies images as Cat or Dog using a CNN model.
A simple web page is created using HTML and CSS to display the project interface.

## Technologies Used
- Python
- TensorFlow / Keras
- Jupyter Notebook
- HTML
- CSS

## Features
- CNN based image classification
- Image prediction using trained model
- Simple front-end web page

## Model File
The trained model is not uploaded due to GitHub file size limits.
The model can be generated by running the notebook.

## Output
The model predicts whether the image is Cat or Dog.

## Future Scope
- Connect web page with backend
- Add more animal classes
